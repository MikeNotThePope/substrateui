Add optional texture utilities: `texture-noise`, `texture-lines`, and `texture-grid`. Pure-CSS background patterns (SVG grain, hairlines, blueprint grid) with zero dependencies. Line and grid inks derive from the foreground token via color-mix, so they adapt to every theme and mode automatically. Opt-in per element; nothing renders differently unless a class is applied.
